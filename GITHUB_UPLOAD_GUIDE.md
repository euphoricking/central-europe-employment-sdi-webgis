# GitHub Upload and GitHub Pages Guide

## Recommended Repository Name

```text
central-europe-employment-sdi-webgis
```

## Upload Instructions

1. Create a new public GitHub repository.
2. Do not upload the ZIP file only.
3. Upload the contents of this project folder so that `index.html` is in the repository root.
4. Your repository should look like this:

```text
index.html
README.md
portfolio_project_text.md
GITHUB_UPLOAD_GUIDE.md
css/
js/
data/
docs/
assets/
```

## Enable GitHub Pages

1. Open the repository on GitHub.
2. Go to **Settings**.
3. Select **Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select branch `main` and folder `/root`.
6. Click **Save**.

Your site will be published at a URL similar to:

```text
https://YOUR-USERNAME.github.io/central-europe-employment-sdi-webgis/
```
